Question-1: What are the different ways to select an element in the DOM?
Here are Some different ways to select an element in the DOM.
  • By ID . Example: document.getElementById("myId")
  • By Class Name . Example: document.getElementsByClassName("myClass")
  • By Tag Name . Example: document.getElementsByTagName("p")
  • By Query Selector . Example: document.querySelector(".myClass")
  • By Query Selector All . Example: document.querySelectorAll(".myClass")
  • By Name Attribute . Example: document.getElementsByName("username")
Question-2: What is the difference between innerHTML, innerText, and textContent ?
Differences between innerHTML, innerText, and textContent:
  • innerHTML . Returns or sets the HTML content inside an element.
  • innerText . Returns or sets only the visible text inside an element.
  • textContent . Returns or sets all text content, including hidden text.
Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single parent element handles events for multiple child elements using event bubbling. Instead of attaching event listeners to each child, we attach a single listener to the parent and use event.target to identify the clicked element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior where an event starts at the target element and propagates upward through its parent elements, all the way to the document root.
